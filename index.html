Description

This page is a directory of tests to show how AngularJS can be used like jQuery by injecting directives by class like a jQuery selector.

Examples are:

  • event: how to bind an event to a DOM element
  • bind: how to bind data to a DOM element
  • update: how to perform an update of a model data through DOM element
  • template: example how to use templates in angular, not comparted vs jQuery because it requires an external plugin
